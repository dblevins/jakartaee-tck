Jakarta Security 1.0 TCK Documentation

The Jakarta Security 1.0 TCK documentation bundle includes the following TCK documentation:

  • The Jakarta Security 1.0 TCK Release Notes provide basic information about the test harness, the test suite and the platforms tested. This document is also included with the TCK itself.

  • The Jakarta Security 1.0 Technology Compatibility Kit User's Guide describes how to install, set up, and run the Jakarta Security 1.0 TCK. It also describes certification and compatibility requirements for the Jakarta Security 1.0 TCK.

  • Javadoc Assertions lists all the javadoc assertions that are tested by the Jakarta Security TCK.

  • Specification Assertions lists all the specification assertions that are tested by the Jakarta Security TCK.

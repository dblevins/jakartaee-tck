Jakarta Security Technology Compatibility Kit, Version 1.0
Release Notes, September 2017

Kit Contents

The Jakarta Security, Version 1.0, Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta Security TCK documentation:

    • These Release Notes can be found in both the TCK bundle
      • If changes or additions are required, you will find them at the Jakarta EE Security Specification web site: https://jakarta-ee/specifications/security/1.0/

       

  • Jakarta Security TCK Signature, API, and specification tests:

    • A signature test that checks that all of the public APIs are supported or defined as specified in the Jakarta Security Version 1.0 implementation under test.

    • API tests for the public APIs and language elements defined by the Jakarta Security 1.0 Specification.

    • Specification tests for the testable assertions required by the Jakarta Security 1.0 Specification.

Platform Notes

The Jakarta Security TCK tests have been built and tested with a Jakarta EE Compatibile Implementation Eclipse GlassFish 5.1 running on the JavaTM Platform, Standard Edition 8 (Java SE 8) platform. If you are using a different Web server, you will need to make adjustments to the procedures outlined in this document.

The Jakarta Security tests run on the following platforms:

  • Oracle Linux 7.1
  • Microsoft Windows 10

Installing, Setting Up, and Running the Jakarta Security TCK

Refer to the Jakarta Security 1.0 Technology Compatibility Kit User's Guide for complete instructions on installing, setting up, and running the Jakarta Security TCK. The online version of the JT Harness version 5.0 documentation is available here.

Guide to the Jakarta EE 8 Platform 8 CTS Documentation

The Jakarta EE 8 Platform 8 CTS documentation includes the following:

  • The Jakarta EE 8 Platform CTS Release Notes provides information about the test harness, the test suite, and the platforms tested.

    The Release Notes may be udpated and released through the Jakarta EE 8 Platform Specification web site: https://jakarta.ee/specifications/full-profile/8.0/ or https://jakarta.ee/specifications/webl-profile/8.0/.

  • The Java Platform Enterprise Edition 8 Compatibility Test Suite User's Guide provides comprehensive user documentation, including instructions for installing and configuring the test suite, and using the JavaTest test harness to execute the tests. The User's Guide also contains the rules you need to pass for certification.

  • The Specification Assertion List lists all the specification assertions that are tested by the Jakarta EE 8 Platform 8 CTS test suite.

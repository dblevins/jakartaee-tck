Jakarta Standard Tag Library Technology Compatibility Kit, Version 1.2
Release Notes, September 2019

Kit Contents

A Jakarta Standard Tag Library, Version 1.2, Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta Standard Tag Library TCK signature, API and specification tests:

    • A signature test that checks that all of the public APIs are supported or defined in the Jakarta Standard Tag Library Version 1.2 implementation that is under test.

    • Specification tests for the testable assertions required by the Jakarta Standard Tag Library Revision 1.2 Specification.

    • API tests for the public APIs and language elements defined by the Jakarta Standard Tag Library 1.2 Specification.

Platform Notes

The Jakarta Standard Tag Library 1.2 TCK tests a Jakarta Standard Tag Library 1.2 Compatible Implementation (CI) and can be run on JavaTM Platform, Standard Edition (Java SE) version 8 on the web container in which you run the tests. See Compatible Implementation for Jakarta Standard Tag Library 1.2 for additional information.

Refer to the A Standard Tag Library for JavaServer PagesTM User's Guide for platform compatibility requirements and notes.

Compatible Implementation for A Standard Tag Library 1.2

The designated Runtime for compatibility testing of implementations based upon the Jakarta Standard Tag Library Specification 1.2 is as follows:

  • Java Standard Edition (Java SE) 8

    •  

  • Eclipse Compatible Implementation (CI) of Jakarta Standard Tag Library 1.2 available as a standalone implementation from Jakarta EE Specifications web site, or included as part of the Jakarta EE 8 compatible implementation, for example Eclipse GlassFish 5.1, or the Eclipse GlassFish 5.1, Web Profile CI.

    Note - Jakarta EE 8 is not required; this TCK only requires a container that supports Jakarta Standard Tag Library 1.2 and its requirements.

  • Jakarta Standard Tag Library TCK tests run on the following platforms:
    • MAC OS X, El Capitan (10.9.5+)
    • Oracle Linux 7.1
    • Windows 10
